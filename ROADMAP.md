# TankFlare Roadmap

This roadmap explains the general direction of TankFlare and the features we would like to build over time.

This is not a strict promise or deadline list. Plans may change as the project grows, contributors join, and better ideas come up.

## Current Focus

A regularly updated list of the features we plan to introduce in the near-future:

- **2D and 3D aquascaping tools**
- **More plant/fish species**
- **More community features**

## Planned Features

A list of features we plan to implement in the future:

- **Saltwater support**

## Contributor Opportunities

Most features require a lot of work! There are a lot of opportunities to help:

- Adding missing plant/fish species
- Improving existing care information
- Fixing inaccurate or outdated data
- Adding images where allowed
- Improving documentation
- Opening issues for missing information or possible mistakes

Please read the [contributing guide](CONTRIBUTING.md) for more information.

## What We Do Not Want

To keep TankFlare helpful and accurate, we do not want:

- Copy-pasted care sheets
- Unsourced claims copied from random websites
- AI-generated care information that has not been checked
- Large unrelated changes in a single pull request
- Guesswork presented as fact

Please read the [Original Content Guidelines](ORIGINAL_CONTENT_GUIDELINES.md) before contributing care information.
